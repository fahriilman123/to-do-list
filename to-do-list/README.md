# Aplikasi To-Do List

## Deskripsi
Aplikasi Sederhana berbasis laravel untuk mencatat tugas harian

##fitur
-Tambah tugas
-Tandau tugas selesai
-Hapus tugas


##struktur folder

-'asset/style,css'

## Cara menjalankan
1 salin folder
2. Jalankan xampp dan buka di "http://localhost/todolist"
